Vue基础-01笔记

vue 介绍

vue 概述

  • 特点：易用、灵活、高效
  • 版本：2.6
  • 作者：尤雨溪
  • 兼容：IE8及以下不兼容

vue使用方法

  1. 引入 vue.js
  2. 创建挂载节点标签
    • 注意：不能再 body 和 html 标签上竞争挂载
  3. 实例化 vue 对象
    • el：挂载节点的选择器
    • data：模板中用到的所有数据

MVVM

  • M model (data里面的数据)
  • V view (视图，就是模板)
  • VM (模拟模型)

vue 指令

  • v-html 渲染内容，可以编译 html 字符串
  • v-text 渲染内容
  • v-cloak 防止闪烁
  • v-pre 跳过编译
  • v-once 一次性绑定
  • v-on 事件绑定
  • v-bind 属性绑定
  • v-model 双向数据绑定

vue 事件

绑定

  1. v-on：事件类型=’事件函数’

  2. @事件类型=’事件函数’

传参

​ @事件类型=’事件函数(实参)’

事件对象

​ @事件类型=’事件函数($event)’

既传参又获取事件对象

​ @事件类型=’事件函数($event，实参)’

事件简写

​ @事件类型=’变量赋值操作’

事件修饰符

修饰符

​ @事件类型.修饰符=’事件函数’

修饰符内容

  • .stop

  • .prevent

  • .self

  • .capture

  • .once

    按键修饰符(一般用于 input 和 textarea)
    • keydown
    • keyup
    • keypress
    • input

vue 样式

class

对象

​ ：class=’{class名1：变量1，class名2：变量2}’

数组

​ ：class=’[变量1，变量2，’class名1’，’class名2’]’

属性绑定

​ ：class=’条件？’class名1’：’class名2’’

style

对象

​ ：style=’{css属性名1：变量1，css属性名2：变量2}’

数组

​ ：style=’[变量1，变量2]’

​ 变量1：{css属性名1：css属性值1}

​ 变量2：{css属性名2：css属性值2}

属性绑定

​ 字符串拼接和模板字符串
